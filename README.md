# mono-dark

A published Omarchy theme that turns the blue dark variant into a pure black
monochrome mode.

## Install

```bash
omarchy-theme-install https://github.com/ryangerardwilson/omarchy-mono-dark-theme.git
```

If you clone it manually:

```bash
git clone https://github.com/ryangerardwilson/omarchy-mono-dark-theme.git ~/.config/omarchy/themes/mono-dark
omarchy-theme-set mono-dark
```

## Included Overrides

- `colors.toml` for Omarchy-generated theme files
- `alacritty.toml`, `btop.theme`, `hyprland.conf`, `mako.ini`, and `tmux.conf`
  for the monochrome UI treatment
- `mono-dark.tmTheme` and `neovim.lua` for editor alignment
- `icons.theme` and `vscode.json` for app-specific theme selection
- `wallpaper.txt`, `wallpaper.svg`, `preview.png`, and `backgrounds/0-mono-dark.png`
